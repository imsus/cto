# CTO Career Lifecycle

Personal work folder for company reporting and career protection.

## Structure

| Folder      | Purpose                                                                   |
| ----------- | ------------------------------------------------------------------------- |
| `strategy/` | KPIs, roadmap, budget, leadership notes (CEO-facing)                      |
| `ic-work/`  | PM-tracked projects + ad-hoc IC work                                      |
| `people/`   | Dewo's 1:1s & growth, peer sync notes                                     |
| `report/`   | Reporting pipeline: impact-log → daily → weekly → quarterly → semi-annual |
| `_private/` | Comp docs, raw 360 feedback (me only)                                     |
| `_media/`   | Screenshots, evidence for all claims                                      |
| `archive/`  | Historical data                                                           |

## Reporting Pipeline

```
impact-log (anytime)
  → kukuruyuk recap (daily, saved)
    → monday checkin (weekly, saved)
      → shimekai (quarterly, slides)
        → executive report (semi-annual, CEO + board)
```

## Quick Start

1. Start logging in `report/impact-log.md` — 5 min per entry, no format stress
2. Use agent skills (`.agents/skills/`) to generate reports at each pipeline step
3. Drop screenshots into `_media/` and reference them by filename

## Key Terms

- **KPI**: derived from company profit goals (no OKRs)
- **Kukuruyuk**: daily all-hands standup
- **Shimekai**: quarterly all-hands townhall
- **PA**: semi-annual performance appraisal (1:1 with CEO)
