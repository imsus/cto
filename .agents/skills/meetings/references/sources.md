# Meetings - Sources & Further Reading

## manager.dev Articles

- [Distracting software engineers](https://newsletter.manager.dev/p/distracting-software-engineers) - flow state, meeting culture fixes, fixed meeting windows, and async defaults

## External Articles

- [Maker's Schedule, Manager's Schedule](https://www.paulgraham.com/makersschedule.html) - Paul Graham; a meeting costs an EM one slot and an engineer the whole maker block
- [How to run better meetings](https://open.substack.com/pub/torstenwalbaum/p/how-to-run-better-meetings) - Torsten Walbaum; four meeting types, scheduling questions, attendance, running, and follow-up
- [Practices to reduce meetings](https://medium.com/illumination/back-to-back-meetings-create-an-illusion-of-productivity-why-the-best-leaders-keep-an-empty-adbb02abdc0f) - proactivity ratio, batching meetings, delegating attendance, and shortening by default
