# People

Managing your reports, plus hiring pipeline.

## Structure

```
people/
  dewo/
    1on1-notes.md   — each entry dated
    growth.md       — trajectory, skills, promotion readiness
  peers/            — 1:1 notes with COO, Finance, Sales, etc.
  pipeline.md       — only when hiring
```

---

## Example: `dewo/1on1-notes.md`

```markdown
# 1:1 Notes — Dewo

## 2026-05-18
- Worked through API pagination refactor — went well
- Brought up wanting more ownership of backend infra
- Next steps: assign infra cost monitoring task

## 2026-05-04
- Checked in on dashboard work — on track
- Feedback: PR descriptions need more context
- Action: I'll share a good example PR
```
