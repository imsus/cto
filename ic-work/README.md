# IC Work

Individual contributor artifacts: what the PM sees, plus everything else.

## Files

| File | Purpose | When |
|------|---------|------|
| `pm-streams.md` | Official PM-tracked work: sync notes, specs, status | During/after each PM sync |
| `unowned.md` | Ad-hoc work: fixes, tooling, exploration, tech debt | As it happens |

---

## Example: `pm-streams.md`

```markdown
# PM-Tracked Work

## Partner Dashboard — PM: [name]

### 2026-05-15 — Sync
- API integration complete ✅
- UI: list view done, detail view in progress
- Need design spec for error states
- Blockers: none

### 2026-05-08 — Sync
- Agreed on scope for v1: read-only dashboard
- Deferred: write operations, export to CSV
- Estimate: June 1 launch, 2-week buffer

## [Project Name] — PM: [name]

### 2026-05-10 — Kickoff
- Goal: reduce checkout errors 50%
- My part: error logging + alerting
- Deadline: May 25
```
